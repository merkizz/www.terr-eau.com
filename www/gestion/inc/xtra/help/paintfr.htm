Utilisation du logiciel "Paint"
 
1 - Ouverture du programme
  Cliquez sur 'Démarrer' puis 'Tous les programmes' puis 'Accessoires' et 'Paint' (vers le bas de la liste).
 
2 - Ouvrir une image
  - Dans le menu, cliquer sur 'Fichier' puis 'Ouvrir'
  - Sélectionnez dans le dossier approprié l'image que vous voulez modifier
 
3 - Modifier l'image
 
  • Réduction de la taille
1 - Verifier la taille de l'image
  • Dans le menu : cliquer sur 'Images' puis 'Attributs'
Les images provenants d'un appareil photo numérique ont en général les tailles standards suivantes :

Largeur x hauteur (en pixel)
2272 x 1704
1600 x 1200
1024 x 768
800 x 600

Selon la taille affichée, nous allons maintenant réduire l'image en largeur et hauteur
(et par conséquent réduire son 'poids')

 
2 - Retailler l'image
  • Dans le menu : cliquer sur 'Images' puis 'Etirer/Incliner'
Nous allons modifier les pourcentages 'Horizontal' et 'Vertical' de manière identique,
afin de respecter les proportions de l'image.
Sachant que beaucoup d'internautes ont une configuration d'écran de 800x600, votre image ne devra pas dépasser la taille de 640x480
(Sur Internet, les tailles standards des photos agrandies sont bien souvent infèrieures)
3 - A ce stade, vous devez enregistrer cette image dans un nouveau dossier, afin de préserver l'image originale.
4 - Vérifier à nouveau la taille de l'image
  • Dans le menu : cliquer sur 'Images' puis 'Attributs'
La 'Taille sur le disque' ne doit pas dépasser 100000 octets (soit 100ko)
Si c'est le cas, vous pouvez la télécharger avec Kaméléon Online
